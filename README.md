# pycobj

Experimental accessing of C data as python objects, based on types from an m2c context header.

## Features

- Memory types
    - [x] RAM dump or other binary file
    - [ ] live dolphin memory
    - [ ] live console memory over network
- Object types
    - [x] Integers
    - [ ] Structs
        - [x] Reading
        - [ ] Writing
    - [ ] Unions
    - [ ] Enums
    - [ ] Pointers
    - [ ] Arrays
        - [x] Reading
        - [ ] Writing
    - [ ] Strings
